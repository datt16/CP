## No.02-B
```c
int n[] = {1980, 1990, 2000, 2010, 2015, 2020, 2300};  
```
のように定義した、任意の西暦年の配列について、
それぞれ**うるう年かどうか**を判定して表示する

```
うるう年は、以下の条件で判別できる
1. 西暦年が4で割り切れる年はうるう年
2. ただし、西暦年が100で割り切れる年は平年（うるう年ではない）
3. ただし、西暦年が400で割り切れる年はうるう年
```
